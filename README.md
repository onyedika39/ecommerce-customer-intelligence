# E-Commerce Customer Intelligence Platform

## Project Overview

This project is an end-to-end data science and AI engineering portfolio project. The goal is to analyze customer behavior, predict customer churn, generate business insights, and eventually deploy the machine learning model as an API.

## Business Problem

E-commerce companies need to understand why customers stop buying. This project uses customer behavior and transaction data to predict customer churn and support better business decisions.

## Project Objectives

- Clean and prepare customer data
- Perform exploratory data analysis
- Build a churn prediction machine learning model
- Evaluate model performance
- Analyze sales and customer behavior using SQL
- Build a Power BI dashboard
- Deploy the trained model using FastAPI, Docker, and Azure
